🌱 FarmTech Solutions – Projeto de Agricultura Digital

Neste projeto, desenvolvi uma aplicação completa utilizando Python e R para simular o gerenciamento de uma fazenda com foco em Agricultura Digital.

Em Python, implementei um sistema interativo via terminal que permite o cadastro e gerenciamento de dados de duas culturas (café e milho). A aplicação realiza o cálculo da área total da fazenda, divisão da área entre culturas, quantidade de ruas de plantio e volume de insumos para pulverização. Também inclui o cálculo da área efetivamente pulverizada, armazenando todas as informações em vetores com funcionalidades de inserção, visualização, atualização e exclusão de dados.

Além disso, os dados processados são exportados para um arquivo CSV, garantindo a integração com a etapa analítica.

Em R, desenvolvi um script que consome esse CSV e realiza análises estatísticas básicas, como cálculo de média e desvio padrão, permitindo validar e interpretar os dados gerados pela aplicação.

O projeto foi estruturado e versionado no GitHub, simulando um ambiente de desenvolvimento organizado e colaborativo. Trata-se de uma aplicação prática que conecta lógica de programação, manipulação de dados e análise estatística aplicada ao contexto do agronegócio. 🚜📊
